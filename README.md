# azure-siem-honeypot
Deployed a live Windows honeypot on Azure exposed to the internet. Configured AMA + DCR pipelines to ingest real-time Windows Security Event logs into Microsoft Sentinel. Built KQL queries and a custom Workbook to visualize global brute-force attack patterns on a live heat map.
Here is the live website: https://ferozahmadkhan.github.io/azure-siem-honeypot/
